# Workspace Utilities & Automation Scripts

Colección de scripts y utilidades en Python para la automatización de tareas de desarrollo, manipulación de datos y gestión de proyectos en el workspace.

## 🛠️ Herramientas Incluidas

* **`ruc_console.py`**: Script de consola para validación y procesamiento de datos RUC (Registro Único de Contribuyentes).

## 🚀 Requisitos e Instalación

* **Entorno**: Linux / Xubuntu
* **Lenguaje**: Python 3.x

```bash
# Ejemplo de ejecución
python3 ruc_console.py
